🚀 REXO AI | ULTRA-CORE V4.1
The Next-Generation Neural Opportunity Pipeline

REXO AI is an advanced, browser-based intelligence tool designed to parse bulk emails and raw datastreams. It automatically extracts high-value opportunities—such as Internships, Jobs, and Scholarships—using a custom Regex-driven Neural Logic that eliminates the need for manual separators or clean data formatting.

🛠 CORE LOGIC & ARCHITECTURE
1. Neural Pattern Splitting
The system bypasses traditional "delimiter-based" parsing. Instead, it identifies the start of a new communication block by detecting:

Header Markers: Subject:, From:, or To:

Structural Patterns: Numerical lists (e.g., 1., 2.) or repeated email address patterns.

Timestamp Detection: Standard date formats frequently used in email logs.

2. Eligibility Scoring Algorithm
Every detected opportunity is put through a Match Index (%) calculation:

Base Intelligence: 25% starting score if identified as an opportunity.

Skill Synergy: +20% for every unique user-defined skill found in the text.

Academic Alignment: +25% if the user’s "Major" matches the document context.

Priority Boost: +10% if a specific deadline or date-sensitive keyword is detected.

3. Persistent Vault (Local Storage)
Built with Data Privacy in mind, the tool utilizes the localStorage API. All "Saved for Later" items are stored directly in your browser. This ensures that your data remains available even after a page refresh or browser restart, without ever leaving your machine.

🌟 KEY FEATURES
⚡ High-Velocity Scanning: An asynchronous engine capable of processing 100+ raw emails in milliseconds.

🔗 Auto-Portal Detection: Automatically crawls the text for URLs and creates a direct "Apply" button within the UI.

💎 Smart Visual Cues: Opportunities with a match score above 60% are automatically highlighted with a Neon Priority Glow.

📱 Premium Dashboard UI: A sleek, dark-themed experience powered by Tailwind CSS and the Outfit typeface.

🔔 Real-time Neural Toasts: Smooth, animated notifications for every vault action (Save/Remove).

🚀 HOW TO DEPLOY
Save the entire source code as an .html file (e.g., rexo_v4.html).

Open the file in any modern web browser (Chrome, Edge, or Brave).

Configure your Neural Profile in the sidebar (Skills & Major).

Paste your raw email dump into the input buffer and click "Execute Scan".
